⚡ ULTRA ENERGIE AUTOMATISERING (iOS + ANDROID)

🌙 iOS NACHTMODUS (20:00 – 08:00)

1. Open: Opdrachten → Automatisering → Persoonlijke automatisering
2. Kies: Tijd van de dag
3. Stel in: 20:00 → dagelijks
4. Zet uit: “Vraag vóór uitvoeren”
Acties toevoegen:
5. Voeg toe: Energiebesparingsmodus → AAN
6. Voeg toe: Focus instellen → Slaap / Niet storen
7. Voeg toe: Helderheid instellen → 30–50%
8. WiFi laten: AAN
9. Mobiel netwerk laten: AAN
10. Bluetooth: optioneel (AAN of UIT)
11. Voeg melding toe: “🌙 Nachtmodus actief”
Focus instellingen (belangrijk):
- Oproepen toestaan: Iedereen of Favorieten
- Herhaalde oproepen: AAN
- Tijdgevoelige meldingen: AAN
👉 Dit houdt je bereikbaar

🔋 iOS NOODMODUS (≤10% batterij)

1. Open: Opdrachten → Automatisering
2. Kies: Batterijniveau → Onder 10%
3. Zet uit: “Vraag vóór uitvoeren”
Acties toevoegen:
4. Energiebesparingsmodus → AAN
5. Helderheid → 10–20%
6. Focus → Niet storen / Persoonlijk
7. WiFi → AAN
8. Mobiel netwerk → AAN
9. Bluetooth → UIT
Focus instellingen:
- Oproepen: Iedereen of Favorieten
- Herhaalde oproepen: AAN
- Belangrijke meldingen: AAN
👉 Bereikbaar blijven is prioriteit

🤖 ANDROID NACHTMODUS (20:00 – 08:00)

1. Open: Bixby Routines / MacroDroid / Tasker
2. Maak nieuwe routine
3. Trigger: Tijd → 20:00 dagelijks
Acties:
4. Energiebesparing → AAN
5. Donkere modus → AAN
6. Helderheid → 20–40%
7. WiFi → AAN
8. Bluetooth → optioneel UIT
9. Sync → beperken (optioneel)

🔋 ANDROID NOODMODUS (≤10% batterij)

1. Trigger: Batterij onder 10%
Acties:
2. Energiebesparing → AAN
3. Helderheid → laag (10–20%)
4. Bluetooth → UIT
5. WiFi → AAN
6. Mobiele data → AAN
Android kan meer automatiseren dan iOS, zoals: - achtergronddata beperken - sync per app uitschakelen - CPU performance modes (afhankelijk toestel)
